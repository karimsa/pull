# @karimsa/pull

Download files using HTTP range transfers.

## Usage

Install via npm - `npm install -g @karimsa/pull` or use via npx `npx @karimsa/pull`.

```bash
$ pull "http://mydomain.com/large-file.zip"
```

To see a help message, use `pull --help`.
